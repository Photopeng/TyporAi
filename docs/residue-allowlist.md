# Residue allowlist

Do not remove these references during terminology cleanup:

- MIT license and upstream attribution, including the Claudian attribution in the README.
- Claude CLI-native `.claude/settings.json`, `.claude/mcp.json`, `.claude/commands`, `.claude/skills`, and `.claude/agents` paths.
- Explicit compatibility migration code and tests for historical settings or sessions.
- Third-party SDK/API names that require their original spelling.

All other user-facing terminology should prefer TyporAi, Typora, workspace, and document.
